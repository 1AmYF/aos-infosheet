# aos-infosheet
Quick overview and basic introduction to the game *Ace of Spades Classic* on a single page, responsive and multi-lingual. Maybe to use as a promotional landing page in the future. [Live Version](https://1amyf.github.io/aos-infosheet)

## Translations
Please feel free to translate the page to a new language, either with a pull request on the `translations.js` file or contact me for alternative ways.
